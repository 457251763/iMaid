## 1. 项目现状分析

### 1.1 现有状态 (v0.1)

| 模块 | 状态 | 说明 |
|------|------|------|
| 项目目录 | ✅ 已完成 | 前后端目录骨架已建立 |
| DESIGN.md | ✅ 已完成 | 完整设计文档 v2.0 |
| 后端脚手架 | ✅ 已完成 | FastAPI 应用入口 |
| 前端脚手架 | ✅ 已完成 | React + Vite + Electron |

### 1.2 技术栈确认

```
前端: Electron + React 18 + Vite + PIXI.js
后端: Python 3.10+ + FastAPI + ChromaDB
通信: WebSocket + REST API + IPC
AI:   DeepSeek / MiniMax
窗口: 悬浮置顶、透明、无边框
```

### 1.3 架构特点

**悬浮窗口模式**:
- 320x400 像素小窗口
- 右下角默认位置
- 始终置顶 (alwaysOnTop)
- 透明背景 (transparent)
- 无边框 (frame: false)
- 不显示在任务栏 (skipTaskbar)
- 可拖拽移动

**语音服务架构**:
- Python 后台进程独立运行
- 通过进程管道通信
- 支持热词唤醒、STT、TTS

- [ ] **3.1.1** 完善 `backend/voice_service.py`
  ```python
  class VoiceService:
      def __init__(self):
          self.wake_word = "小 Maid"
          self.is_listening = False
          self.running = True
      
      def handle_command(self, command):
          if command == "wake":
              self.wake_up()
          elif command == "sleep":
              self.go_to_sleep()
      
      def wake_up(self):
          self.is_listening = True
          self.send_to_renderer({"type": "voice_wake"})
      
      def go_to_sleep(self):
          self.is_listening = False
          self.send_to_renderer({"type": "voice_sleep"})
  ```

```
<类型>(<模块>): <描述>
```
