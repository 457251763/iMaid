## 📝 技术实现细节

### DraggableContainer 组件 API
```jsx
<DraggableContainer
  className="custom-class"
  enableDrag={true}
  dragHandleClass="drag-handle"
  bounds={{ minX: 0, maxX: 1000, minY: 0, maxY: 800 }}
>
  {children}
</DraggableContainer>
```

### 拖拽事件流程
1. 用户在拖拽手柄上按下鼠标（mousedown）
2. 记录初始鼠标位置和窗口位置
3. 用户移动鼠标（mousemove）
4. 计算鼠标移动距离，更新窗口位置
5. 调用 `window.electronAPI.moveWindow(x, y)` 移动窗口
6. 用户释放鼠标（mouseup），结束拖拽

### IPC 通信
- **Preload API**：`window.electronAPI.getWindowPosition()` 和 `window.electronAPI.moveWindow(x, y)`
- **Main Process**：处理窗口位置获取和设置
